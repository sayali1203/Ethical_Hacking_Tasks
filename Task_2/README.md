Task: Task 02 – Network Scanning & Service Enumeration

Objective

The objective of this task is to understand the scanning and enumeration phase of ethical hacking using Nmap. The task involves identifying active hosts, open ports, running services, service versions, and operating system information using safe and authorized methods.

Tools Used

- Nmap
- Npcap
- Windows Command Prompt
- Microsoft Word
- GitHub


Commands Used

nmap --version
nmap localhost
nmap -sV localhost
nmap -O localhost

Task Summary

- Installed and verified Nmap.
- Scanned the local machine (localhost).
- Identified open ports and running services.
- Performed service version detection.
- Detected the operating system.
- Researched common network ports and their security risks.
- Prepared a professional scan report with observations and recommendations.

Learning Outcomes

- Learned the basics of network scanning.
- Understood service enumeration using Nmap.
- Identified open ports and active services.
- Learned the importance of operating system detection.
- Improved knowledge of common network ports and their security risks.
- Understood the importance of scanning only authorized systems.
