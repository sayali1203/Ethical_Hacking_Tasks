Name: Sayali Patil

Objective

The objective of this task is to understand the first phase of Ethical Hacking, known as Reconnaissance (Information Gathering). The task involves collecting publicly available information about a target website using only passive reconnaissance techniques.

Target Website

Website Name: Savitribai Phule Pune University

Website URL: https://unipune.ac.in

Tools Used

- WHOIS Lookup (whois.com)
- MXToolbox
- WhatRuns
- SecurityHeaders.com
- Web Browser (Google Chrome)

Folder Contents

- Report
- Screenshots
- Research Notes
- README.md

Learning Outcomes

- Performed passive reconnaissance on a public website.
- Collected WHOIS information.
- Identified DNS records.
- Identified website technologies.
- Analyzed HTTP Security Headers.
- Examined robots.txt and sitemap.xml.
- Understood the importance of ethical information gathering before penetration testing.
